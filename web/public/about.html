♻ Proyecto ECO-SORT v2.0

ECO-SORT es una máquina autónoma de clasificación de residuos que combina hardware IoT, visión artificial con GCP Vertex AI y una plataforma web gamificada.

◈ Arquitectura del Sistema

🔧

Capa Física

Raspberry Pi Zero W como cerebro central, Arduino Uno para control de servos MG996R, cámara USB 640×480 y pantalla touch HDMI.

Capa Nube

GCP Vertex AI con GPU NVIDIA T4 ejecuta el modelo SigLIP2 en tiempo real. Inferencia en menos de 2 segundos con 91.28% de precisión.

Explicación completa del proyecto

1

El usuario se acerca a la máquina

La pantalla touch muestra un código único (ECO-XXX). El usuario lo ingresa en la app web para vincular su sesión.

2

Coloca el objeto frente a la cámara

La cámara captura un frame en 640×480px. La Raspberry Pi lo envía vía HTTPS al endpoint de GCP Vertex AI.

3

El modelo SigLIP2 clasifica el material

En menos de 2 segundos, el modelo identifica la categoría con 91.28% de precisión y devuelve JSON con categoría y confianza.

4

El servo dirige el objeto

La Raspberry Pi envía el comando serial al Arduino. El servo MG996R gira y dirige el objeto al contenedor correcto.

5

Se otorgan puntos XP en tiempo real

Socket.io transmite el resultado a la plataforma web. El usuario ve la clasificación, confianza y XP ganados en vivo.
